# PDF Link - Universal File to PDF Converter

A beautiful, modern web application that converts any file type to professional PDF format while preserving the original formatting and structure.

## ✨ Features

- **Universal File Support** - Upload any file type (text, code, images, documents, binary files)
- **Original Format Preservation** - Maintains exact formatting, spacing, and structure
- **Beautiful UI** - Modern, responsive design with smooth animations
- **Professional PDF Output** - Clean, readable PDFs with proper styling
- **Drag & Drop Interface** - Easy file upload with visual feedback
- **Real-time Processing** - Fast file conversion with progress indicators

## 🚀 Supported File Types

### Text & Code Files
- Documents: `.txt`, `.md`, `.rtf`
- Code files: `.js`, `.py`, `.java`, `.cpp`, `.html`, `.css`
- Data files: `.json`, `.csv`, `.xml`, `.yaml`
- Configuration files: `.ini`, `.conf`, `.toml`

### Media & Binary Files
- Images: `.jpg`, `.png`, `.gif`, `.svg`
- Documents: `.pdf`, `.docx`, `.xlsx`
- Media files: `.mp4`, `.mp3`, `.avi`
- Archives: `.zip`, `.tar`, `.rar`

## 🛠️ Tech Stack

- **React** with TypeScript
- **Tailwind CSS** for styling
- **jsPDF** for PDF generation
- **Lucide React** for icons
- **Vite** for build tooling

## 🏃‍♂️ Getting Started

1. Clone the repository:
```bash
git clone https://github.com/T2-Astra/PDF_Link.git
cd PDF_Link
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open [http://localhost:5173](http://localhost:5173) in your browser

## 📦 Build for Production

```bash
npm run build
```
